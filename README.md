## 📖 Description

This repository contains Python programs developed as part of Assignment 1 for the Scripting for Hackers subject. Each program demonstrates fundamental Python concepts such as user input, arithmetic operations, conditional statements, and simple calculations.

---

## 📂 Programs Included

### 1. Compound Interest Calculator
- Calculates compound interest using principal, rate, and time.

### 2. Average of Five Numbers
- Calculates the average of five user-entered numbers.

### 3. Simple Login System
- Verifies username and password using conditional statements.

### 4. Student Marks Calculator
- Calculates total marks, average, and percentage for five subjects.

### 5. Day of the Week
- Displays the day based on the user's choice (1–7).

### 6. Swap Two Numbers
- Swaps two numbers without using a third variable.

### 7. Arithmetic Operations
Performs:
- Addition
- Subtraction
- Multiplication
- Division
- Floor Division
- Modulus
- Exponent
